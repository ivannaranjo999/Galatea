# Galatea Task Manager
## Dependencies
- ncurses

## Compile and install

To build Galatea, run:
```
make
```

To install Galatea in `~/.local/bin`, run:
```
make install
```

To uninstall Galatea from `~/.local/bin`, run:
```
make uninstall
```

Ensure that `~/.local/bin` is present in your `PATH` environment variable
